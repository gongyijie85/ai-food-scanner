# Issue tracker

This project uses **GitHub Issues** in the repository `gongyijie85/ai-food-scanner`.

## Creating an issue

Use the `gh` CLI:

```bash
gh issue create --title "..." --body "..."
```

## Issue conventions

- Title: short and actionable.
- Body: include reproduction steps, expected vs actual behavior, and relevant screenshots/logs.
- Labels: apply triage labels via `gh issue edit <number> --add-label <label>`.
