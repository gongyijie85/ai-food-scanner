# Triage labels

| Canonical role | Label | Meaning |
|---|---|---|
| Needs evaluation | `needs-triage` | Maintainer has not yet evaluated the issue. |
| Waiting on reporter | `needs-info` | More information is needed from the issue reporter. |
| Ready for AI | `ready-for-agent` | Fully specified; an AFK agent can pick it up without human context. |
| Ready for human | `ready-for-human` | Requires human implementation or decision. |
| Won't fix | `wontfix` | Will not be actioned. |
