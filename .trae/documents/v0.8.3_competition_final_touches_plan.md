## Assumptions & Decisions

- **DEBUG 彻底移除**：为确保生产环境安全，直接删除 DEBUG 相关代码而非条件判断
- **TTS 使用浏览器原生**：不引入外部 TTS 服务（如 Edge TTS Python 库），保持零依赖
- **按钮位置调整**：通过 CSS 调整间距而非改动 HTML 结构，减少风险
