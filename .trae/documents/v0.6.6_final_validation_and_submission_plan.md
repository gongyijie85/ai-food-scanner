### Task 1：修复移动端底部导航 bug

**Files:**
- Modify: `components/navigation.py:7-12`（移除未使用的图标 import）
- Modify: `components/navigation.py:23-28`（将 tabs 改为 2 元组）
- Modify: `components/navigation.py:36`（循环解包改为 `(page, label)`）
- Modify: `components/navigation.py:47-48`（按钮 label 改为纯文字 `label`）

**原因：** v0.6.3 清理 SVG 图标时，移动端底部导航的 tabs 结构未同步调整，导致运行时解包错误。

**修改后代码（精确替换）：**

```python
# 原 import 块
from components.icons import (
    _ICON_CAMERA,
    _ICON_HISTORY,
    _ICON_HOME,
    _ICON_PROFILE,
)

# 替换为：底部导航不再使用 SVG 图标
from components.icons import _ICON_EMPTY
```

```python
# 原 tabs 定义
tabs = [
    ("home", _ICON_HOME, "首页"),
    ("scan", _ICON_CAMERA, "扫描"),
    ("history", _ICON_HISTORY, "历史"),
    ("profile", _ICON_PROFILE, "我的"),
]

# 替换为
tabs = [
    ("home", "首页"),
    ("scan", "扫描"),
    ("history", "历史"),
    ("profile", "我的"),
]
```

```python
# 原循环
for col, (page, label) in zip(cols, tabs):

# 保持不变（现在 tabs 是 2 元组，解包正确）
for col, (page, label) in zip(cols, tabs):
```

```python
# 原按钮 label
if st.button(
    f"{icon}\n{label}",
    width="stretch",
    key=f"mobile_nav_{page}",
):

# 替换为
if st.button(
    label,
    width="stretch",
    key=f"mobile_nav_{page}",
):
```

- [ ] **Step 1.1:** 打开 `components/navigation.py`，按上述精确替换修改。
- [ ] **Step 1.2:** 检查 `_ICON_EMPTY` 是否真实存在于 `components/icons.py`；若不存在，则保留任意一个最小 import 占位或删除整个 import 块。
